# Enabler Assignment

## Setup

### Backend
cd server
npm install
node app.js

### Frontend
cd client
npm install
npm run dev

## Features
- Upload 3 CSV files
- Validation (missing columns, empty data)
- Linking via Component_ID and Product_SKU
- Aggregated sales
- Shows:
  - Preview
  - Validation errors
  - Matched data
  - Unmatched data
